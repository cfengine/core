## Download
```
git clone https://github.com/olehermanse/agentsay.git
cd agentsay
```

## Install
```
make install
```

## CLI usage
```
agentsay Hello, agent
```

## Python usage
```Python
import agentsay
agentsay.speak("Hello, agent")
```
